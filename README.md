# ☕ Cafe Finder

A Cafe Finder web application built using JavaScript, Tailwind CSS, Leaflet.js, OpenStreetMap, and Overpass API.

## Features

* Detects user's current location
* Displays nearby cafes on an interactive map
* Search cafes by city
* Sidebar listing nearby cafes
* Responsive UI using Tailwind CSS

## Technologies Used

* HTML
* CSS
* Tailwind CSS
* JavaScript
* Leaflet.js
* OpenStreetMap
* Overpass API

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:npx serve .

4. Open the generated localhost URL in your browser
